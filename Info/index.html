Metodología SCRUM

Es un tipo de método de trabajo ágil en grupo, dentro de ellos tienen distintos roles, entre ellos product owner, scrum master y el scrum team.

Roles

Product owner: Es el que tiene contacto directo con el cliente para saber las necesidades que tienen este respecto al programa que se esta realizando, este es el encargado de revisar que las actas, daily meetings 

Scrum Master: Por medio de este el team transmite cualquier problema al Product owner para que este le informe al cliente, este es el encargado del las daily meetings, y actas a los que se compromete cada miembro del team. 

Scrum Team: El team solo toca codigo, y informa cualquier inconveniente al Scrum Master
